# Regresní analýza

## Vložení dat

Data lze do tabulky zadat ručně kliknutím do příslušného políčka tabulky a napsáním/vložením čísla. Aplikace považuje za desetinnou čárku znaky `,` a `.`. Stisknutím klávesy `Enter` dojde automaticky k přeskočení do dalšího políčka tabulky. Tímto způsobem je možné bez použití myši postupně zadávat dvojice $(x_1, y_1), (x_2, y_2), \ldots$

Aplikace umožňuje kromě ručního zadávání také nahrát data ve formátech `.txt`, `.csv`, `.tsv` a `.dat` kliknutím na tlačítko <svg viewBox="0 0 24 24" width="15" height="15" style="vertical-align:-3px" fill="none" stroke="currentColor" stroke-width="1.8" stroke-linecap="round" stroke-linejoin="round"><path d="m6 14 1.5-2.9A2 2 0 0 1 9.24 10H20a2 2 0 0 1 1.94 2.5l-1.54 6a2 2 0 0 1-1.95 1.5H4a2 2 0 0 1-2-2V5a2 2 0 0 1 2-2h3.9a2 2 0 0 1 1.69.9l.81 1.2a2 2 0 0 0 1.67.9H18a2 2 0 0 1 2 2v2"/></svg>`načíst soubor`. Autodetekce oddělovače sloupců rozpoznává `,` (čárku), `;` (středník), tabulátor a mezeru; obsahuje-li řádek mezery, má mezera přednost a čárka se považuje za desetinnou (české zvyklosti). Aplikace má ve výchozím stavu osy pojmenované jako $x$ a $y$ (políčka nad tabulkou). Pokud nahraný soubor obsahuje záhlaví (první řádek s názvy sloupců), aplikace ho automaticky rozpozná a použije jako názvy os. V obou případech je možné názvy libovolně přepsat. Od verze `1.11` obsahuje aplikace také možnost <svg viewBox="0 0 24 24" width="15" height="15" style="vertical-align:-3px" fill="none" stroke="currentColor" stroke-width="1.8" stroke-linecap="round" stroke-linejoin="round"><path d="m6 14 1.5-2.9A2 2 0 0 1 9.24 10H20a2 2 0 0 1 1.94 2.5l-1.54 6a2 2 0 0 1-1.95 1.5H4a2 2 0 0 1-2-2V5a2 2 0 0 1 2-2h3.9a2 2 0 0 1 1.69.9l.81 1.2a2 2 0 0 0 1.67.9H18a2 2 0 0 1 2 2v2"/></svg>`načíst pokročilé`. V tomto případě je možné vkládat data jak po sloupcích, tak i po řádcích, a to i v případě, kdy soubor obsahuje více než 2 řádky/sloupce. Toto tlačítko tedy otevře jednoduchého průvodce, kde je nutné zakliknout, jestli jsou data ve formátu řádků nebo sloupců a poté klikem na záhlaví příslušného řádku/sloupce vybrat který bude brán jako $x$ a který jako $y$.

Sloupce tabulky je možné prohodit tlačítkem <svg viewBox="0 0 24 24" width="15" height="15" style="vertical-align:-3px" fill="none" stroke="currentColor" stroke-width="1.8" stroke-linecap="round" stroke-linejoin="round"><path d="m16 3 4 4-4 4"/><path d="M20 7H4"/><path d="m8 21-4-4 4-4"/><path d="M4 17h16"/></svg>`prohodit x ↔ y`, tím dojde k prohození os $(x_1 \leftrightarrow y_1,\ x_2 \leftrightarrow y_2,\ \ldots)$. Tlačítkem <svg viewBox="0 0 24 24" width="15" height="15" style="vertical-align:-3px" fill="none" stroke="currentColor" stroke-width="1.8" stroke-linecap="round" stroke-linejoin="round"><path d="M15.2 3a2 2 0 0 1 1.4.6l3.8 3.8a2 2 0 0 1 .6 1.4V19a2 2 0 0 1-2 2H5a2 2 0 0 1-2-2V5a2 2 0 0 1 2-2z"/><path d="M17 21v-7a1 1 0 0 0-1-1H8a1 1 0 0 0-1 1v7"/><path d="M7 3v4a1 1 0 0 0 1 1h7"/></svg>`uložit data` dojde k uložení dat tabulky ve formátu `.txt` a tlačítkem <svg viewBox="0 0 24 24" width="15" height="15" style="vertical-align:-3px" fill="none" stroke="currentColor" stroke-width="1.8" stroke-linecap="round" stroke-linejoin="round"><path d="M3 6h18"/><path d="M19 6v14a2 2 0 0 1-2 2H7a2 2 0 0 1-2-2V6m3 0V4a2 2 0 0 1 2-2h4a2 2 0 0 1 2 2v2"/><line x1="10" y1="11" x2="10" y2="17"/><line x1="14" y1="11" x2="14" y2="17"/></svg>`smazat data` k jejich odstranění. Jsou-li v tabulce data a uživatel vloží nová, dojde k přepsání starých dat.

## Více sad dat

Aplikace umí pracovat až s 5 sadami dat najednou. Sady se přepínají záložkami v levém sloupci, nová sada se přidá tlačítkem `+`. Každá sada má vlastní tabulku, typ regrese, výsledky i barvu, kterou se odlišuje v grafu (barevná tečka na záložce odpovídá barvě bodů v grafu). V grafu se vždy zobrazují všechny sady, které obsahují data, regresní parametry v pravém sloupci ale patří vždy právě aktivní (vybrané) sadě. Jakmile mají data alespoň 2 sady, objeví se navíc tlačítko **SVG**`uložit vše`, které uloží společný graf všech sad do jednoho souboru.

## Vyhodnocení dat

Symbolem ✅ jsou označeny ty řádky tabulky, na kterých se provádí regresní analýza. Při spuštění aplikace jsou takto označeny všechny řádky, přičemž je možné klikem myši na tento symbol libovolné řádky z účasti v regresi vyloučit. Stejným symbolem v záhlaví tabulky je možné najednou odznačit a zpětně označit všechny řádky najednou, případně zrušit provedení výběru bodů. Dále je možné provést výběr bodů pro regresi pomocí tlačítka <svg viewBox="0 0 24 24" width="15" height="15" style="vertical-align:-3px" fill="none" stroke="currentColor" stroke-width="1.8" stroke-linejoin="round" stroke-dasharray="3 3"><rect x="4" y="4" width="16" height="16" rx="1"/></svg>`výběr oblasti`. V takovém případě dojde automaticky ke zrušení označení všech bodů tabulky. Označení bodů se zde provede podržením levého tlačítka myši a současným tažením přes zvolené body. Tyto body se po provedení označení označí zpět v tabulce. Označené body jsou v grafu zobrazeny plným tvarem a neoznačené body stejným tvarem, ale prázdným (jen obrysem).

Kliknutím na tlačítko <svg viewBox="0 0 24 24" width="15" height="15" style="vertical-align:-3px" fill="none" stroke="currentColor" stroke-width="1.8" stroke-linecap="round" stroke-linejoin="round"><polyline points="3 17 9 11 13 15 21 7"/><polyline points="15 7 21 7 21 13"/></svg>`typ regrese` se zobrazí rozbalovací nabídka, ve které je možné vybrat tvar fitované funkce. Po výběru se její obecný předpis vypíše v poli dole pod grafem. Zapnutí a vypnutí zobrazení fitované funkce v grafu se provede tlačítkem <svg viewBox="0 0 24 24" width="13" height="13" style="vertical-align:-2px" fill="currentColor" stroke="none"><polygon points="6 3 20 12 6 21 6 3"/></svg>`spustit analýzu`. V pravém sloupci se zobrazí parametry fitování a v oblasti dole pod grafem se pod obecným předpisem fitované funkce zobrazí stejná s konkrétními parametry. Tlačítko **95%**`interval spolehlivosti` v grafu zobrazí 95% interval spolehlivosti. Tlačítko **SVG**`uložit graf` (a při více sadách **SVG**`uložit vše`) otevře nabídku se dvěma možnostmi: `Rychlé uložení (SVG)` uloží graf tak, jak je, a `Pokročilý průvodce…` otevře plnohodnotný editor exportu (viz samostatná kapitola níže). Výsledný SVG je vektorový — lze ho upravovat v Illustratoru/Inkscape i vložit do Wordu/PowerPointu beze ztráty kvality při zvětšení.

Nad grafem se zobrazuje legenda, ve které mohou být celkem 4 pojmy: `Data` označují body, na kterých je prováděna regrese, `fit` je nafitovaná funkce, `Vyloučeno (X)` jsou body vyloučené z regrese včetně jejich počtu v závorce a `IS 95 %` je 95% interval spolehlivosti. Kliknutím na libovolný údaj dojde ke zrušení zobrazování příslušných dat v grafu. Rozsahy os se nastavují automaticky, ale je možné je nastavit manuálně prostřednictvím políček `x min`, `x max`, `y min` a `y max`. Kliknutím na tlačítko <svg viewBox="0 0 24 24" width="15" height="15" style="vertical-align:-3px" fill="none" stroke="currentColor" stroke-width="1.8" stroke-linecap="round" stroke-linejoin="round"><path d="M3 12a9 9 0 0 1 15-6.7L21 8"/><path d="M21 3v5h-5"/><path d="M21 12a9 9 0 0 1-15 6.7L3 16"/><path d="M8 21H3v-5"/></svg>`reset os` dojde k opětovnému přepnutí na automatické nastavení.

## Typy regresí

Nabídka fitovaných funkcí postupně narostla. K dispozici jsou: lineární $y = ax + b$, exponenciální $y = a\cdot e^{bx}$, polynomická 2. stupně $y = ax^2 + bx + c$, logaritmická $y = a\cdot\ln(x) + b$, Gaussova křivka (a také součet dvou nebo tří Gaussových křivek — u každého píku se zobrazí i jeho FWHM, tedy šířka v polovině výšky), lomená lineární funkce, Fourierova řada a vlastní rovnice. U všech parametrů se vedle hodnoty vypisuje i jejich nejistota (směrodatná odchylka) a kvalita fitu $R^2$.

**Fourierova řada** má vlastní panel nastavení: počet harmonických složek se volí posuvníkem (případně ho aplikace zvolí automaticky podle kvality fitu) a periodu je možné buď nechat najít automaticky, nebo zadat ručně políčkem `Vlastní perioda`. Tlačítko `periodogram` otevře graf kvality fitu ($R^2$) pro jednotlivé testované periody — vybraná perioda je v něm zvýrazněná, takže je hned vidět, jestli automatika trefila správný vrchol. Tlačítko `rezidua` (dostupné pro všechny typy regresí) otevře graf reziduí, tedy odchylek naměřených bodů od fitované funkce — užitečné pro posouzení, jestli je zvolený model vhodný.

## Vlastní rovnice

Volba `Vlastní rovnice` v nabídce typů regrese otevře okno, kde je možné napsat libovolný předpis funkce, např. `a*sin(b*x+c)+d` nebo `Vmax*x/(Km+x)`. Podporované funkce jsou `sin, cos, tan, exp, log, sqrt, abs` a základní operátory `+ - * / ^`. Všechna písmena kromě `x` aplikace automaticky považuje za hledané parametry a nafituje je. Rovnice se při psaní průběžně vykresluje v matematické sazbě, takže překlepy jsou vidět okamžitě. Povedenou rovnici je možné tlačítkem <svg viewBox="0 0 24 24" width="15" height="15" style="vertical-align:-3px" fill="none" stroke="currentColor" stroke-width="1.8" stroke-linecap="round" stroke-linejoin="round"><path d="M15.2 3a2 2 0 0 1 1.4.6l3.8 3.8a2 2 0 0 1 .6 1.4V19a2 2 0 0 1-2 2H5a2 2 0 0 1-2-2V5a2 2 0 0 1 2-2z"/><path d="M17 21v-7a1 1 0 0 0-1-1H8a1 1 0 0 0-1 1v7"/><path d="M7 3v4a1 1 0 0 0 1 1h7"/></svg> uložit pod vlastním názvem do rychlého výběru — knihovna uložených rovnic se pak nabízí přímo v rozbalovací nabídce typů regrese a přežije i zavření prohlížeče.

## Matematické nástroje

Tlačítko <svg viewBox="0 0 24 24" width="15" height="15" style="vertical-align:-3px" fill="none" stroke="currentColor" stroke-width="1.8" stroke-linecap="round" stroke-linejoin="round"><circle cx="9" cy="12" r="6"/><circle cx="15" cy="12" r="6"/></svg>`nástroje` otevře panel se třemi nástroji, které pracují s nafitovanými funkcemi. Každý nástroj má vlastní přepínač zapnuto/vypnuto, panel jde schovat na lištu (analýza zůstane běžet) nebo úplně zavřít, čímž se všechny nástroje vypnou.

**Kombinace funkcí** vyžaduje alespoň 2 nafitované regrese. Z rozbalovacích nabídek se vybere funkce A, funkce B a operace ($+$, $-$, $\times$) — výsledná zkombinovaná funkce se vykreslí do grafu a její předpis i parametry (včetně nejistot) se zobrazí pod grafem a v panelu.

**Integrál** spočítá určitý integrál $\int_a^b f(x)\,\mathrm{d}x$ zvolené nafitované funkce. Meze $a$ a $b$ se zadávají v panelu, integrovaná plocha se v grafu vyšrafuje a výsledek se zobrazí včetně odhadu nejistoty (vychází z nejistot regresních parametrů). Výpočet probíhá numericky Simpsonovým pravidlem.

**Derivace** vykreslí tečnu ke zvolené nafitované funkci v bodě dotyku $x_0$ a zobrazí hodnotu směrnice $f'(x_0)$. Bod dotyku je možné plynule posouvat posuvníkem (tečna se v grafu pohybuje v reálném čase), nebo zadat přesně číselně.

Zapnuté nástroje se ukládají do projektu a jsou-li aktivní, přenesou se i do exportu `uložit vše` (do exportu jednoho grafu se nekreslí, aby v něm nepřekážely).

## Pokročilý průvodce exportem

Volba `Pokročilý průvodce…` (v nabídce tlačítek **SVG**) otevře editor, ve kterém se graf před uložením upraví přesně podle potřeby — s živým náhledem, kde je každá změna okamžitě vidět. Průvodce existuje ve dvou variantách: pro aktivní sadu dat (`uložit graf`) a pro všechny sady najednou (`uložit vše`).

V levém panelu se postupně nastavuje: **Záhlaví** (nadpis grafu a jeho velikost), **Popisek osy X** a **Popisek osy Y** (text, velikost, zarovnání), **Osy a legenda** (velikost číslic na osách a písma legendy, barva pozadí a vlastní rozsahy os — pokud by nějaká kreslená data padla mimo zadaný rozsah, průvodce na to upozorní), **Sady dat** (pro každou sadu barva a tvar bodů — kruh, trojúhelník, obrácený trojúhelník, čtverec, kosočtverec — velikost bodů, barva, tloušťka a případné čárkování regresní křivky) a **Položky legendy** (každou položku lze přejmenovat nebo úplně skrýt). Značky v legendě vždy přesně odpovídají tomu, co je nakreslené v grafu, včetně barev a tvarů upravených v průvodci.

Nadpis i popisky os je možné v náhledu jednoduše **přetahovat myší** na libovolné místo — kolem grafu je k tomu velkorysý pracovní okraj. U popisků os jsou navíc tlačítka `Zarovnat na střed` a `Zarovnat na okraj`, která popisek umístí na obvyklou pozici jedním klikem. Náhled jde přibližovat a oddalovat tlačítky `−`/`+` v pravém horním rohu (klik na procenta vrátí 100 %). Při uložení se výsledný obrázek **automaticky ořízne** přesně podle skutečně umístěného obsahu, žádné zbytečné okraje.

Každý textový popisek (nadpis, osy i položky legendy) má přepínač **TeX**, který otevře editor pro sazbu v (La)TeXové syntaxi s živým náhledem. Funguje matematika mezi `$...$` (např. `Rychlost $v$ v m$\cdot$s$^{-1}$`), celý popisek jako centrovaná rovnice v `$$...$$` a textové příkazy `\emph{šikmo}`, `\textbf{tučně}`, `\texttt{strojopis}`, `\textsc{kapitálky}`, `\underline{podtržení}`. Takto vysázené popisky se do uloženého SVG vloží věrně, včetně správných fontů — soubor vypadá stejně jako náhled i po otevření na jiném počítači.

Veškerá nastavení průvodce se pamatují (přežijí zavření prohlížeče a ukládají se i do souboru projektu), takže příští export stejného grafu je otázkou jednoho kliknutí. Tlačítko `↺ Obnovit výchozí` vrátí všechna nastavení průvodce na výchozí hodnoty. Okno se zavírá tlačítkem, klávesou `Esc` nebo klikem mimo něj.
